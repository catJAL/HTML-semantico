HTML Semântico

Tags que usamos na construção do site

<p> <header> <main> <section> <article> e <footer>

Explicação e como utilizar as Tags principais do HTML semântico

<header>

Ela é utilizada para se fazer cabeçalhos, nela se pode inserir tags como as de <h1> ao <h6>.

<section>

A tag <section> serve para determinar uma seção onde se tem um tópico e um subtópico.

<article>

A tag <article> serve para seperar dois tópicos em um site, sendo uma tag muito utilizada em blogs e para criação de tópicos

<nav>

A tag <nav> é utilizada para fazer uma listagem de links da web, usando como atributos as tags <ul> e a <li>

<aside>

A tag <aside>, é utilizada quando precisamos criar um conteúdo de apoio/adicional ao conteúdo principal

<main>

A tag <main>, especifica o conteúdo principal do site, e consequentemente dando mais ênfase naquele conteúdo
